# Система учёта товаров (обувь)

## Запуск

1. **БД:** создайте БД в PostgreSQL и выполните скрипт `DB.sql` (или запустите `python create_db.py`).
2. **Настройки:** при необходимости задайте в `App/config.py` или переменные окружения: `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, `DB_PASSWORD`.
3. **Зависимости:** `pip install -r requirements.txt`
4. **Приложение:** `python main.py` (команды — в файле `script`)

## Роли (тестовые пользователи)

- Логин `admin`, пароль `123` — администратор (добавление/редактирование/удаление товаров, поиск, фильтр, сортировка).
- Логин `manager`, пароль `123` — менеджер (просмотр товаров, поиск, фильтр по поставщику, сортировка по количеству).
- Логин `client`, пароль `123` — клиент (просмотр товаров).
- Кнопка «Войти как гость» — просмотр без авторизации.

## Структура

- **App/** — конфиг, БД, окна (Login, Main, Orders, OrderForm, ProdForm, Card). Разметка загружается из **UI/*.ui**.
- **UI/** — файлы разметки Qt Designer (только .ui).
- **logic/** — авторизация, список/редактирование товаров и заказов.
- **main.py** — точка входа. Импорт из Excel: `python import_to_db.py` (xlsx в папке `import/`).

## Изображения товаров

- Сохраняются в папку `product_images`, размер 300×200. Заглушка: `resources/picture.png` (создаётся при первом запуске).
