## Заказ: несколько товаров (Qt Designer)

В `ui/order_form.ui` уже настроено:

- подпись **«Позиции заказа»** и поле **`article_edit`** типа **QPlainTextEdit** (высота ~88px) — удобно вставлять столбцы из Excel;
- **placeholder** с подсказкой по формату;
- **`lbl_article_hint`** — серый поясняющий текст на две колонки (word wrap).

Дополнительно в дизайнере можно: изменить высоту `article_edit`, шрифт подсказки, текст `lbl_article_hint`.

## Импорт из Excel (как раньше в проекте)

Файлы из методички клади в **`resources/`**:

| Файл | Назначение |
|------|------------|
| `user_import.xlsx` | пользователи |
| `Tovar.xlsx` | товары |
| `Заказ_import.xlsx` | заказы + позиции в той же строке: **«Строка заказа»** / **«Артикул заказа»** (артикулы через запятую, как в форме) или **«Номер товара»** + **«Количество»** |
| `Пункты выдачи_import.xlsx` | адреса ПВЗ (1-й столбец; в заказе — номер строки или текст адреса) |

```bash
pip install pandas openpyxl
python import_to_db.py
```

Колонки в Excel — те же, что в старом скрипте (Логин, ФИО, Наименование товара, Дата заказа, …). Позиции заказа формируются скриптом из строки `Заказ_import` (текст артикулов или «Номер товара»). `Код для получения` → `receiver_code`.
