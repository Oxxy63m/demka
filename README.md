# Система учёта товаров

## Быстрый старт

1. Установить Python 3 и PostgreSQL. Выполнить: `pip install -r requirements.txt`
2. Создать БД и таблицы: `python create_db.py` (один раз)
3. При необходимости загрузить данные из Excel: положить файлы в папку из `App/config.py` (DATA_DIR), затем `python import_to_db.py`
4. Запуск программы: `python main.py` или двойной щелчок по `запуск_программы.bat`

## Структура проекта

- **main.py** — запуск приложения
- **create_db.py** — создание БД и таблиц (запускать первым)
- **import_to_db.py** — импорт из Excel в БД
- **DB.sql** — скрипт создания таблиц (выполняется через create_db.py)
- **App/config.py** — настройки (пути, БД). Менять под свой компьютер
- **App/** — окна и работа с БД (Login, Main, Orders, Cart, ProdForm, OrderForm, db)
- **ui/** — разметка окон (.ui)
- **logic/** — логика входа, каталога, заказов, форм

Настройки БД и папки с данными — в `App/config.py`.
