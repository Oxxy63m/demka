# Система учёта товаров (обувь)

Подробная пошаговая инструкция для запуска и изучения — в файле **ИНСТРУКЦИЯ.md** (удобно даже без опыта в Python).

---

## Быстрый старт

1. Установить Python 3, PostgreSQL, выполнить: `pip install -r requirements.txt`
2. Создать БД: `python create_db.py`
3. (По желанию) Загрузить данные: положить Excel в папку из `App/config.py` → `python import_to_db.py`
4. Запуск программы: `python main.py` (или двойной щелчок по `запуск_программы.bat`)

## Структура проекта (для запоминания)

- **main.py** — запуск приложения.
- **create_db.py** — создание базы и таблиц (запускать первым).
- **import_to_db.py** — импорт из Excel в БД.
- **DB.sql** — скрипт создания таблиц (выполняется через create_db.py).
- **App/config.py** — настройки (пути, пароль БД). Менять под свой компьютер.
- **App/** — окна и работа с БД (Login, Main, Orders, Cart и др.).
- **ui/** — разметка окон (.ui).
- **logic/** — логика входа, списков товаров/заказов, сохранения.

Настройки БД и папки с данными — в `App/config.py`.
