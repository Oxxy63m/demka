# Система учёта товаров

## Быстрый старт

1. Установить Python 3 и PostgreSQL. Выполнить: `pip install -r requirements.txt`
2. Создать БД и таблицы: один раз выполнить SQL из `DB.sql` (через pgAdmin / psql) **или** использовать код из `create_db.txt`.
3. При необходимости импортировать данные:
   - Вариант 1: из Excel через скрипт `import_to_db.py` (файлы в папке `resources`, см. `App/config.py`).
   - Вариант 2: через CSV и pgAdmin 4 по памятке `импорт_через_CSV.txt`.
4. Запуск программы: `python main.py`.

## Структура проекта

- `main.py` — запуск приложения.
- `DB.sql` — схема БД в третьей нормальной форме.
- `create_db.txt` — код-пример создания БД и применения `DB.sql` (для справки).
- `import_to_db.py` — импорт данных из Excel в БД.
- `import_to_db.txt` — тот же код импорта в виде текстовой шпаргалки.
- `импорт_через_CSV.txt` — пошаговая инструкция CSV‑импорта через pgAdmin 4.
- `script` — короткая памятка по запуску и импорту.
- `App/` — настройки (`config.py`), окна и работа с БД.
- `ui/` — файлы интерфейса Qt Designer (`*.ui`).
- `logic/` — вспомогательная бизнес‑логика форм и списков.
- `resources/` — иконки, плейсхолдер, фото товаров, Excel/CSV‑файлы.

Настройки подключения к БД и пути к ресурсам находятся в `App/config.py`.
