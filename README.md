# Система учёта товаров (обувь)

## Запуск

1. Создайте БД в PostgreSQL (например, `demka_shoes`) и выполните скрипт `database/schema.sql` в PgAdmin или psql.
2. Настройте при необходимости параметры в `config.py` или переменные окружения: `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, `DB_PASSWORD`.
3. Установите зависимости: `pip install -r requirements.txt`
4. Запуск: `python main.py`

## Роли (тестовые пользователи из schema.sql)

- Логин `admin`, пароль `123` — администратор (добавление/редактирование/удаление товаров, поиск, фильтр, сортировка).
- Логин `manager`, пароль `123` — менеджер (просмотр товаров, поиск, фильтр по поставщику, сортировка по количеству).
- Логин `client`, пароль `123` — клиент (просмотр товаров).
- Кнопка «Войти как гость» — просмотр товаров без авторизации.

## Изображения товаров

- Сохраняются в папку `product_images` в корне приложения, размер при сохранении — 300×200 пикселей.
- Заглушка при отсутствии фото: `resources/picture.png` (создаётся при первом запуске, если отсутствует).
