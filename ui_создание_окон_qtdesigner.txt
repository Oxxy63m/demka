МЕТОДИЧКА: КАК С НУЛЯ СДЕЛАТЬ ТОЧНО ТАКОЙ ЖЕ UI В QT DESIGNER (ПОШАГОВО)

Правила (выполняйте строго):
1) В Designer НЕТ «QVBoxLayout/QHBoxLayout» как виджетов. Используйте Layouts:
   - Vertical layout, Horizontal layout, Grid layout, Form layout.
2) Layout ставится так: выделили область/виджеты → ПКМ → Lay out → нужный layout.
3) Если layout не ставится: ПКМ → Lay out → Break Layout → поставить заново.
4) Имена (objectName) задавайте ТОЧНО как ниже (Property Editor → objectName).
5) Файлы сохраняйте ТОЧНО по путям: папка `ui`, имена как ниже.

ПОРЯДОК СОЗДАНИЯ ФАЙЛОВ
1) `ui/login.ui`
2) `ui/main.ui`
3) `ui/product_item.ui`
4) `ui/cart.ui`
5) `ui/orders_list.ui`
6) `ui/order_form.ui`
7) `ui/product_form.ui`

========================================
1) `ui/login.ui`
========================================
1. File → New → Dialog.
2. Выделите диалог (клик по пустому месту) → ПКМ → Lay out → Lay Out Vertically.
3. Выделите диалог и задайте свойства:
   - objectName: `LoginWindow`
   - windowTitle: `Вход в систему`
   - minimumSize: width 440, height 320
4. Добавьте QLabel (Display Widgets → Label). Поставьте самым верхним.
   - objectName: `lbl_logo`
   - minimumSize: 120×120
   - maximumSize: 150×150
   - scaledContents: true
   - alignment: AlignCenter
   - text: очистить (пусто)
   В Object Inspector: выделите item этого QLabel в layout → alignment: AlignHCenter.
5. Добавьте QLabel:
   - objectName: `lbl_title`
   - text: `Авторизация`
   - alignment: AlignCenter
   - wordWrap: true
6. Добавьте QLabel:
   - objectName: `lbl_login`
   - text: `Логин:`
   - wordWrap: true
7. Добавьте QLineEdit (Input Widgets → Line Edit):
   - objectName: `login_edit`
   - placeholderText: `Введите логин`
8. Добавьте QLabel:
   - objectName: `lbl_password`
   - text: `Пароль:`
   - wordWrap: true
9. Добавьте QLineEdit:
   - objectName: `password_edit`
   - echoMode: Password
   - placeholderText: `Введите пароль`
10. Добавьте 2 кнопки в один ряд:
   10.1 Добавьте 2 QPushButton (Buttons → Push Button) подряд внизу.
   10.2 Выделите ОБЕ кнопки → ПКМ → Lay out → Lay Out Horizontally.
   10.3 Задайте свойства кнопок:
       - первая: objectName `btn_login`, text `Войти`
       - вторая: objectName `btn_guest`, text `Войти как гость`
11. Save As → `ui/login.ui`.

========================================
2) `ui/main.ui`
========================================
1. File → New → Main Window.
2. Выделите окно и задайте свойства:
   - objectName: `ProductListWindow`
   - windowTitle: `Список товаров`
   - minimumSize: 800×500
3. Выделите centralWidget → ПКМ → Lay out → Lay Out Vertically.

Шапка (одна строка):
4. Добавьте в centralWidget: QLabel, QPushButton, QPushButton, QPushButton, Horizontal Spacer, QLabel (именно в этом порядке).
5. Выделите эти 6 элементов → ПКМ → Lay out → Lay Out Horizontally.
6. Задайте свойства:
   - QLabel: objectName `lbl_title`, text `Список товаров`, wordWrap true
   - QPushButton: objectName `btn_orders`, text `Заказы`
   - QPushButton: objectName `btn_cart`, text `Заказ`
   - QPushButton: objectName `btn_logout`, text `Выход`
   - Spacer: оставить как есть
   - QLabel: objectName `lbl_user`, text `Гость`, wordWrap true

Строка фильтров (одна строка):
7. Добавьте: QLabel, QLineEdit, QLabel, QComboBox, QLabel, QComboBox (в этом порядке).
8. Выделите 6 элементов → ПКМ → Lay out → Lay Out Horizontally.
9. Свойства:
   - lbl_search: text `Поиск:`, wordWrap true
   - search_edit: placeholderText `Поиск по названию, артикулу...`
   - lbl_supplier: text `Поставщик:`, wordWrap true
   - supplier_combo: objectName `supplier_combo`
   - lbl_sort: text `Сортировка по количеству:`, wordWrap true
   - sort_combo: objectName `sort_combo`
   Не забудьте objectName у QLabel: `lbl_search`, `lbl_supplier`, `lbl_sort`.

Строка с кнопкой добавления:
10. Добавьте QPushButton.
11. Выделите кнопку → ПКМ → Lay out → Lay Out Horizontally.
12. Свойства: objectName `btn_add`, text `Добавить товар`.

Список карточек:
13. Добавьте Scroll Area (Containers → Scroll Area).
14. Свойства Scroll Area:
    - objectName: `cards_scroll`
    - widgetResizable: true
    - horizontalScrollBarPolicy: ScrollBarAlwaysOff
15. Кликните по внутреннему виджету Scroll Area (контейнер внутри) и задайте:
    - objectName: `cards_container`
16. Выделите `cards_container` → ПКМ → Lay out → Lay Out Vertically.
17. В Object Inspector выделите layout внутри `cards_container` и задайте:
    - objectName: `cards_layout`
    - alignment: AlignTop
18. Save As → `ui/main.ui`.

3) `ui/product_item.ui`
========================================
1. File → New → Widget.
2. В Object Inspector выберите корневой виджет и в свойствах смените class на Frame (Containers → Frame) ИЛИ создайте Frame и сделайте его корнем (проще: сразу File → New → Widget, затем в Object Inspector заменить QWidget на QFrame через Promote/пересоздание). В итоге корень должен быть Frame.
3. Свойства корневого Frame:
   - objectName: `ProductCard`
   - minimumSize: width 400, height 200
   - frameShape: Box
   - lineWidth: 1
4. Выделите Frame → ПКМ → Lay out → Lay Out Horizontally.

Левая колонка (фото):
5. Добавьте QLabel.
6. Свойства:
   - objectName: `photo_label`
   - minimumSize: 300×200
   - alignment: AlignCenter

Центральная колонка (текст с прокруткой):
7. Добавьте Scroll Area.
8. Свойства Scroll Area:
   - objectName: `scroll_center`
   - widgetResizable: true
   - frameShape: NoFrame
   - horizontalScrollBarPolicy: ScrollBarAlwaysOff
9. Внутренний виджет scroll area:
   - objectName: `center_widget`
10. Выделите `center_widget` → ПКМ → Lay out → Lay Out Vertically.
11. Выделите layout внутри `center_widget` и задайте:
   - objectName: `center_layout`
   - spacing: 0
   - leftMargin: 4, topMargin: 2, rightMargin: 4, bottomMargin: 2
12. Внутрь `center_widget` добавьте QLabel (в таком порядке), у всех включите wordWrap true:
   - `lbl_header`
   - `lbl_desc_title`
   - `lbl_desc`
   - `lbl_manufacturer`
   - `lbl_supplier`
   - `lbl_price`
   - `lbl_unit`
   - `lbl_stock`

Правая колонка (скидка и кнопки):
13. Добавьте Widget (Containers → Widget).
14. Свойства:
   - objectName: `right_column`
   - minimumSize: width 180, height 0
15. Выделите right_column → ПКМ → Lay out → Lay Out Vertically.
16. Выделите layout внутри right_column и задайте objectName: `vl_actions`.
17. Добавьте QLabel:
   - objectName: `discount_label`
   - minimumSize: 160×50
   - alignment: AlignCenter
   - wordWrap: true
18. Добавьте QPushButton:
   - objectName: `btn_order`
   - text: `В заказ`
19. Добавьте QPushButton:
   - objectName: `btn_delete`
   - text: `Удалить`
20. Save As → `ui/product_item.ui`.

4) `ui/cart.ui`
========================================
1. File → New → Dialog.
2. Выделите диалог → ПКМ → Lay out → Lay Out Vertically.
3. Свойства диалога:
   - objectName: `CartDialog`
   - windowTitle: `Оформление заказа`
   - minimumSize: 720×480
4. Добавьте QLabel:
   - objectName: `lbl_title`
   - text: `Оформление заказа`
   - wordWrap: true
5. Добавьте Table Widget (Item Widgets → Table Widget):
   - objectName: `table_items`
   - columnCount: 4
   - заголовки колонок (по порядку): `Товар`, `Кол-во`, `Цена`, `Сумма`

6. Добавьте внизу 2 блока в один ряд:
   6.1 Добавьте контейнеры: сначала Layout для формы слева и Layout для итога справа.
   6.2 Проще сделать так: добавьте Widget (контейнер) и ещё один Widget рядом → выделите оба → Lay Out Horizontally.

Левый блок (форма):
7. В левый контейнер добавьте 4 виджета (Label+ComboBox, Label+DateEdit), затем выделите их → Lay Out in a Form.
8. Свойства:
   - lbl_pickup: text `Пункт выдачи`, wordWrap true
   - cb_pickup: objectName `cb_pickup`
   - lbl_delivery: text `Дата выдачи/доставки`, wordWrap true

Правый блок (итого):
9. В правый контейнер добавьте QLabel, QLabel, Vertical Spacer (в этом порядке) → выделите → Lay Out Vertically.
10. Свойства:
   - lbl_total_title: text `Итого:`, wordWrap true
   - lbl_total: text `0.00 ₽`, wordWrap true

Кнопки:
11. Добавьте внизу: PushButton, Horizontal Spacer, PushButton, PushButton → выделите 4 элемента → Lay Out Horizontally.
12. Свойства:
   - btn_remove: text `Удалить позицию`
   - btn_cancel: text `Назад`
   - btn_confirm: text `Оформить`
13. Save As → `ui/cart.ui`.
