МЕТОДИЧКА: КАК С НУЛЯ СДЕЛАТЬ ТОЧНО ТАКОЙ ЖЕ UI В QT DESIGNER (ПОШАГОВО)

Правила (выполняйте строго):
1) В Designer НЕТ «QVBoxLayout/QHBoxLayout» как виджетов. Используйте Layouts:
   - Vertical layout, Horizontal layout, Grid layout, Form layout.
2) В твоей версии Designer «Компоновка» часто неактивна, пока на форме нет ни одного виджета.
   Поэтому: СНАЧАЛА добавь виджеты → ПОТОМ выдели их/контейнер → ПКМ → Компоновка → нужный пункт.
3) Компоновка ставится так: выделили виджеты (или контейнер, если он уже содержит виджеты) → ПКМ → Компоновка → нужный пункт.
   - «Скомпоновать по вертикали» = Vertical layout
   - «Скомпоновать по горизонтали» = Horizontal layout
   - «Скомпоновать по сетке» = Grid layout
   - «Скомпоновать в два столбца» = Form layout
4) Если компоновка не ставится (пункты серые): либо на форме ещё нет виджетов, либо не выделены виджеты/контейнер, либо уже есть компоновка.
   Делайте так:
   - добавьте хотя бы 1 виджет
   - выделите контейнер (диалог/centralWidget/виджет) или группу виджетов
   - ПКМ → Компоновка → Удалить компоновщик
   - затем снова ПКМ → Компоновка → нужный пункт
5) Имена (objectName) задавайте ТОЧНО как ниже (Property Editor → objectName).
6) Файлы сохраняйте ТОЧНО по путям: папка `ui`, имена как ниже.
7) Как выделять “диалог/окно/контейнер”, если не получается:
   - В левом окне Object Inspector кликните по верхнему элементу (например QDialog/QMainWindow),
   - убедитесь, что в Property Editor меняются свойства именно у него,
   - после этого ставьте layout или меняйте windowTitle/minimumSize.
8) Как добавлять виджеты “в нужное место”:
   - Перетаскивайте виджеты на форму в рабочей области (в центр),
   - после добавления проверяйте порядок в Object Inspector (сверху вниз/слева направо),
   - если порядок не тот — перетащите виджет выше/ниже в Object Inspector.

9) Важно про layout’ы:
   - Тип layout (сетка/вертикаль/горизонталь/2 столбца) можно выбирать как удобно.
   - Главное, чтобы на форме были НУЖНЫЕ ВИДЖЕТЫ и их `objectName` совпадали с кодом.
   - Поэтому после каждого окна есть блок «ПРОВЕРЬ ПЕРЕД СОХРАНЕНИЕМ».

========================================
1) `ui/login.ui`
========================================
1. File → New → Dialog.
2. Выделите диалог (Object Inspector → верхний элемент QDialog) и задайте свойства:
   - objectName: `LoginWindow`
   - windowTitle: `Вход в систему`
   - minimumSize: width 440, height 320
3. Добавьте виджеты на форму сверху вниз (пока без компоновки, просто расположи примерно как нужно):
   - Label (логотип)
   - Label (заголовок)
   - Label (Логин:)
   - Line Edit (логин)
   - Label (Пароль:)
   - Line Edit (пароль)
   - Push Button (Войти)
   - Push Button (Войти как гость)
4. Сделайте компоновку:
   4.1 Выделите ДВЕ кнопки → ПКМ → Компоновка → Скомпоновать по горизонтали.
   4.2 Выделите ВСЕ элементы на форме (Ctrl+A) → ПКМ → Компоновка → Скомпоновать по вертикали.
5. Настройте свойства виджетов.
6. Label (логотип) (должен быть первым в вертикальной компоновке):
   - objectName: `lbl_logo`
   - minimumSize: 120×120
   - maximumSize: 150×150
   - scaledContents: true
   - alignment: AlignCenter
   - text: очистить (пусто)
   В Object Inspector: выделите item этого QLabel в layout → alignment: AlignHCenter.
7. Label (заголовок):
   - objectName: `lbl_title`
   - text: `Авторизация`
   - alignment: AlignCenter
   - wordWrap: true
8. Label:
   - objectName: `lbl_login`
   - text: `Логин:`
   - wordWrap: true
9. Line Edit:
   - objectName: `login_edit`
   - placeholderText: `Введите логин`
10. Label:
   - objectName: `lbl_password`
   - text: `Пароль:`
   - wordWrap: true
11. Line Edit:
   - objectName: `password_edit`
   - echoMode: Password
   - placeholderText: `Введите пароль`
12. Кнопки (в нижней строке):
    - первая: objectName `btn_login`, text `Войти`
    - вторая: objectName `btn_guest`, text `Войти как гость`
13. Save As → `ui/login.ui`.
ПРОВЕРЬ ПЕРЕД СОХРАНЕНИЕМ (`ui/login.ui`):
- Корень (QDialog): objectName `LoginWindow`, windowTitle `Вход в систему`, minimumSize 440×320
- Label: `lbl_logo`
- Label: `lbl_title`
- Label: `lbl_login`
- Line Edit: `login_edit`
- Label: `lbl_password`
- Line Edit: `password_edit` (echoMode Password)
- Push Button: `btn_login`
- Push Button: `btn_guest`

   - Push Button
   - Push Button
   - Push Button
   - Horizontal Spacer (Spacers → Horizontal Spacer)
   - Label

6. В Object Inspector переименуйте получившийся горизонтальный layout в `head` (выделите layout и задайте objectName).
7. Задайте свойства виджетов в этой строке:
   - QLabel: objectName `lbl_title`, text `Список товаров`, wordWrap true
   - QPushButton: objectName `btn_orders`, text `Заказы`
   - QPushButton: objectName `btn_cart`, text `Заказ`
   - QPushButton: objectName `btn_logout`, text `Выход`
   - Spacer: оставить как есть
   - QLabel: objectName `lbl_user`, text `Гость`, wordWrap true

Строка фильтров (одна строка):
8. Добавьте элементы СТРОКИ ФИЛЬТРОВ (ниже шапки):
   - Label
   - Line Edit
   - Label
   - Combo Box
   - Label
   - Combo Box
9. Выделите 6 элементов → ПКМ → Компоновка → Скомпоновать по горизонтали.
10. В Object Inspector переименуйте этот layout в `row_filter`.
11. Свойства:
   - lbl_search: text `Поиск:`, wordWrap true
   - search_edit: placeholderText `Поиск по названию, артикулу...`
   - lbl_supplier: text `Поставщик:`, wordWrap true
   - supplier_combo: objectName `supplier_combo`
   - lbl_sort: text `Сортировка по количеству:`, wordWrap true
   - sort_combo: objectName `sort_combo`
   Не забудьте objectName у QLabel: `lbl_search`, `lbl_supplier`, `lbl_sort`.

Строка с кнопкой добавления:
12. Добавьте Push Button (ниже фильтров) → выделите кнопку → ПКМ → Компоновка → Скомпоновать по горизонтали.
13. В Object Inspector переименуйте layout этой строки в `row_buttons`.
14. Свойства кнопки: objectName `btn_add`, text `Добавить товар`.

Список карточек:
15. Добавьте Scroll Area (Containers → Scroll Area) (ниже кнопки добавления).
16. Свойства Scroll Area:
    - objectName: `cards_scroll`
    - widgetResizable: true
    - horizontalScrollBarPolicy: ScrollBarAlwaysOff
17. Кликните по внутреннему виджету Scroll Area (контейнер внутри) и задайте:
    - objectName: `cards_container`
18. ВАЖНО: если `cards_container` пустой, «Компоновка» может быть серой.
    Сделайте так:
    - добавьте ВНУТРЬ `cards_container` любой временный Label (любой текст),
    - выделите `cards_container` → ПКМ → Компоновка → Скомпоновать по вертикали,
    - удалите временный Label (он больше не нужен).
19. В Object Inspector выделите layout внутри `cards_container` и задайте:
    - objectName: `cards_layout`
    - alignment: AlignTop
20. Теперь выделите ВЕСЬ centralWidget (Object Inspector → `centralwidget`) → ПКМ → Компоновка → Скомпоновать по вертикали.
    (Если пункты серые — выделите все элементы внутри centralWidget (Ctrl+A) и сделайте «Скомпоновать по вертикали».)
21. Save As → `ui/main.ui`.
ПРОВЕРЬ ПЕРЕД СОХРАНЕНИЕМ (`ui/main.ui`):
- Корень (QMainWindow): objectName `ProductListWindow`, windowTitle `Список товаров`, minimumSize 800×500
- Label: `lbl_title`
- Push Button: `btn_orders`
- Push Button: `btn_cart`
- Push Button: `btn_logout`
- Label: `lbl_user`
- Label: `lbl_search`
- Line Edit: `search_edit`
- Label: `lbl_supplier`
- Combo Box: `supplier_combo`
- Label: `lbl_sort`
- Combo Box: `sort_combo`

- Scroll Area: `cards_scroll`
- Внутри Scroll Area: `cards_container`
- Layout внутри `cards_container`: objectName `cards_layout`, alignment AlignTop

3) `ui/product_item.ui`
========================================
1. File → New → Widget.
2. Сразу задайте “тип формы” как Frame:
   - меню Form → Form Settings…
   - в поле “Form class” выберите `QFrame`
   - нажмите OK
   (В итоге корневой элемент станет Frame, и его можно настраивать.)
3. Свойства корневого Frame:
   - objectName: `ProductCard`
   - minimumSize: width 400, height 200
   - frameShape: Box
   - lineWidth: 1
4. Выделите Frame → ПКМ → Компоновка → Скомпоновать по горизонтали.

Левая колонка (фото):
5. Добавьте QLabel.
6. Свойства:
   - objectName: `photo_label`
   - minimumSize: 300×200
   - alignment: AlignCenter

Центральная колонка (текст с прокруткой):
7. Добавьте Scroll Area.
8. Свойства Scroll Area:
   - objectName: `scroll_center`
   - widgetResizable: true
   - frameShape: NoFrame
   - horizontalScrollBarPolicy: ScrollBarAlwaysOff
9. Внутренний виджет scroll area:
   - objectName: `center_widget`
10. Выделите `center_widget` → ПКМ → Компоновка → Скомпоновать по вертикали.
11. Выделите layout внутри `center_widget` и задайте:
   - objectName: `center_layout`
   - spacing: 0
   - leftMargin: 4, topMargin: 2, rightMargin: 4, bottomMargin: 2
12. Внутрь `center_widget` добавьте QLabel (в таком порядке), у всех включите wordWrap true:
   - `lbl_header`
   - `lbl_desc_title`
   - `lbl_desc`
   - `lbl_manufacturer`
   - `lbl_supplier`
   - `lbl_price`
   - `lbl_unit`
   - `lbl_stock`

Правая колонка (скидка и кнопки):

   - minimumSize: width 180, height 0
15. Выделите right_column → ПКМ → Компоновка → Скомпоновать по вертикали.
16. Выделите layout внутри right_column и задайте objectName: `vl_actions`.
17. Добавьте QLabel:
   - objectName: `discount_label`
   - minimumSize: 160×50
   - alignment: AlignCenter
   - wordWrap: true
18. Добавьте QPushButton:
   - objectName: `btn_order`
   - text: `В заказ`
19. Добавьте QPushButton:
   - objectName: `btn_delete`
   - text: `Удалить`
20. Save As → `ui/product_item.ui`.
ПРОВЕРЬ ПЕРЕД СОХРАНЕНИЕМ (`ui/product_item.ui`):
- Корень (QFrame): objectName `ProductCard`, frameShape Box, lineWidth 1
- Label (фото): `photo_label`

4) `ui/cart.ui`
========================================
1. File → New → Dialog.
2. Выделите диалог (Object Inspector → верхний элемент QDialog) и задайте свойства:
   - objectName: `CartDialog`
   - windowTitle: `Оформление заказа`
   - minimumSize: 720×480
3. СНАЧАЛА добавьте виджеты на форму (пока без компоновки):
   - Label (заголовок)
   - Table Widget
   - Widget (левый блок)
   - Widget (правый блок)
   - Push Button (Удалить позицию)
   - Horizontal Spacer
   - Push Button (Назад)
   - Push Button (Оформить)
4. Теперь сделайте компоновку:
   4.1 Выделите левый и правый Widget → ПКМ → Компоновка → Скомпоновать по горизонтали.
   4.2 Выделите 4 элемента кнопок (PushButton + Spacer + PushButton + PushButton) → ПКМ → Компоновка → Скомпоновать по горизонтали.
   4.3 Выделите ВСЕ элементы на форме (Ctrl+A) → ПКМ → Компоновка → Скомпоновать по вертикали.
5. Настройте свойства.
6. Заголовок (Label):
   - objectName: `lbl_title`
   - text: `Оформление заказа`
   - wordWrap: true
7. Table Widget:
   - objectName: `table_items`
   - columnCount: 4
   - заголовки колонок (по порядку): `Товар`, `Кол-во`, `Цена`, `Сумма`

6. Сделайте строку “Пункт выдачи/Дата” слева и “Итого” справа:
   6.1 Добавьте Widget (Containers → Widget) — это будет ЛЕВЫЙ блок.
   6.2 Добавьте второй Widget — это будет ПРАВЫЙ блок.
   6.3 Выделите оба Widget → ПКМ → Компоновка → Скомпоновать по горизонтали.
   6.4 В Object Inspector переименуйте этот горизонтальный layout в `hl_row`.

Левый блок (форма):
7. Выделите ЛЕВЫЙ блок (первый Widget) и добавьте в него 4 виджета в таком порядке:
   - Label, Combo Box
   - Label, Date Edit
   Затем выделите эти 4 виджета → ПКМ → Компоновка → Скомпоновать в два столбца.
   В Object Inspector переименуйте form layout в `formLayout`.
8. Свойства:
   - lbl_pickup: text `Пункт выдачи`, wordWrap true
   - cb_pickup: objectName `cb_pickup`
   - lbl_delivery: text `Дата выдачи/доставки`, wordWrap true
   - de_delivery: objectName `de_delivery`, calendarPopup true

Правый блок (итого):
9. Выделите ПРАВЫЙ блок (второй Widget) и добавьте:
   - Label
   - Label
   - Vertical Spacer (Spacers → Vertical Spacer)
   Выделите эти 3 элемента → ПКМ → Компоновка → Скомпоновать по вертикали.
   В Object Inspector переименуйте вертикальный layout в `vl_total`.
10. Свойства:
   - lbl_total_title: text `Итого:`, wordWrap true
   - lbl_total: text `0.00 ₽`, wordWrap true

Кнопки:
11. Добавьте внизу (под строкой `hl_row`): PushButton, Horizontal Spacer, PushButton, PushButton.
    Выделите 4 элемента → ПКМ → Компоновка → Скомпоновать по горизонтали.
    В Object Inspector переименуйте этот layout в `hl_buttons`.
12. Свойства:
   - btn_remove: text `Удалить позицию`
   - btn_cancel: text `Назад`
   - btn_confirm: text `Оформить`
13. Save As → `ui/cart.ui`.
ПРОВЕРЬ ПЕРЕД СОХРАНЕНИЕМ (`ui/cart.ui`):
